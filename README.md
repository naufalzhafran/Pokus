# Pokus - A Simple and Light Pomodoro Timer

Pokus is a minimalistic and lightweight Pomodoro Timer built with [Next.js 15](https://nextjs.org/) and styled using [ShadCN UI](https://shadcn.dev/). It is designed to help you focus on your tasks and manage your time effectively.


## 🚀 Features

- 🕒 **Pomodoro Timer**: Stay productive with the classic 25/5-minute Pomodoro technique.
- 🌟 **Lightweight and Fast**: Built on the latest Next.js for optimal performance.
- 🎨 **Beautiful UI**: Styled using ShadCN for a modern and sleek design.
- 🔄 **Customizable Sessions**: Adjust work and break durations as per your needs. (Coming Soon)


## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/)
- **UI Library**: [ShadCN UI](https://shadcn.dev/)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (integrated with ShadCN UI)
